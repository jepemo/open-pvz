Tower Defense Game Engine inspired by "Plantz Vz Zombiez", implemented in C with differents bindings for other languages.

## Installation
### Linux (Ubuntu)

Install compiler and lua headers

```bash
apt-get install gcc
```
### MacOS
Install [brew](https://brew.sh/) 

Install lua: 

```bash
brew install lua
```
